Machine learning

do you have any recommendations for books on machine learning that I could read?

Fantastic question, there are a lot of books out there that can help you transition into machine learning. Try and see many different opinions and perspectives. A good starting point:

  1. Elements of Statistical Learning
  2. Probabilistic Machine Learning
  3. Understanding Machine Learning: From Theory to Algorithms
  4. Foundations of Machine Learning
  5. Stanford course on Machine Learning (YouTube link)

Deep Learning

do you have any recommendations on where should I start understanding deep learning?

Fantastic question! Deep learning is still constantly evolving and changing, and doing research in it is to always stay on the cutting edge

  1. Neural Networks and Deep Learning
  2. Deep Learning
  3. Introduction to Deep Learning tutorials (YouTube playlist)
  4. Stanford Computer Vision (YouTube playlist)
  5. Deep Learning at Oxford (YouTube playlist)
  6. The Principles of Deep Learning Theory

Radboud Related Teaching/Advising

I wanna do a bachelors or masters in AI, what should I go over?

Study your math and fundamentals properly. Come well prepared on your linear algebra, calculus, optimization, statistics and probability. If you have the time it's super worthwhile for AI students to go over Information theory.
